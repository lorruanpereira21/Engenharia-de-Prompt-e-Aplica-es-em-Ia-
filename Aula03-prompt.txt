## 📌 Conceito
Programação é o ato de **transformar problemas em soluções computacionais** por meio de **algoritmos** escritos em uma linguagem que o computador entende.  
No curso de ADS, isso significa aprender a **analisar sistemas**, **modelar soluções** e **codificar** para que elas funcionem de forma eficiente.  

---

## 🖥️ Exemplo em Python
Um exemplo simples que ilustra o conceito de programação é calcular a média de notas de um aluno:

```python
# Conceito de programação aplicado em Python

# Passo 1: Definir o problema
# Queremos calcular a média de notas de um aluno

# Passo 2: Criar um algoritmo (sequência lógica)
notas = [8.5, 7.0, 9.2, 6.8]  # lista de notas

# Passo 3: Implementar em uma linguagem de programação
media = sum(notas) / len(notas)

# Passo 4: Mostrar o resultado
print("A média do aluno é:", media)
```

---

## 🔎 Explicação
- **Análise:** Identificamos o problema (calcular média).  
- **Desenvolvimento:** Criamos um algoritmo (somar notas e dividir pela quantidade).  
- **Sistemas:** Implementamos em Python, que é uma linguagem de programação muito usada em ADS.
